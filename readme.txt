To run the program just use the command  

phyton tictactoe.py type

type is the type of IA just put the letter acording to the next table

m for minimax
e for expectimax
a for alfa-beta pruning

Ex:
phyton tictactoe.py a 			to execute with alfa-beta pruning

Wait until the window appear and just click in the cell you want to mark


Note:

You move first with the "X"
The IA moves with the "O"
